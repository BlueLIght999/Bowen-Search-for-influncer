## 判定标准

- 爆点综合分 = hookStrength * 0.3 + scriptQuality * 0.2 + emotionalRhythm * 0.2 + differentiation * 0.3
- 综合分 >= 75：高爆点潜力
- 综合分 55-74：中等爆点潜力
- 综合分 < 55：低爆点潜力

## 传播触发条件（满足 3 条以上才可能爆发）

1. 前三秒钩子强度 >= 70（决定完播率基线）
2. 脚本结构完整，有明确反转点（决定收藏率）
3. 情绪曲线有至少 1 个峰值和 1 个谷值（决定评论率）
4. 差异化分 >= 60（决定是否被算法识别为新内容）
5. 结尾有明确的互动引导（决定转发率）

## 正例

钩子75 + 脚本70 + 情绪80 + 差异化65 = 72.5（中等偏高），满足条件 1/2/3/5 → 有爆发可能

## 反例

钩子40 + 脚本50 + 情绪40 + 差异化30 = 39（低），仅满足条件 5 → 不可能爆发

## 评分映射

- 综合分 >= 75 且满足 4+ 条件：viralPotential +25
- 综合分 55-74 且满足 3 条件：viralPotential +12
- 综合分 < 55：viralPotential -20

## 适用场景

- 选题评估：生成候选方向后计算爆点潜力
- 内容评估：对已完成视频做爆点诊断
- 改造建议：针对不满足的条件给出具体优化方向
