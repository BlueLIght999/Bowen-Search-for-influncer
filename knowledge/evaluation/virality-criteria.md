## 框架来源

适配自开源项目 AI-Youtube-Shorts-Generator 的 VIRALITY_CRITERIA 八维评估体系，
原项目用于从长视频中提取高传播力片段，博闻将其本地化为中文短视频内容评估框架。

## 8 维传播力信号（按影响力排序）

### 1. 钩子时刻 (HOOK MOMENTS)

制造即时好奇心的陈述，让观众在前 3 秒停下划屏。

**识别特征**：
- "秘密是..." / "没人告诉你..." / "我之前完全搞错了..."
- 暗示有反常识信息即将揭示
- 制造信息差，让观众觉得"只有这里才有答案"

**中文平台典型句式**：
- "99%的人都不知道..." / "我试了100次才发现..."
- "这个行业最大的谎言是..." / "别再做XX了，真正有效的是..."

**评分映射**：命中钩子时刻 → viralPotential +15

### 2. 情绪峰值 (EMOTIONAL PEAKS)

真实的情感爆发——惊讶、大笑、愤怒、脆弱、兴奋，而非表演式情绪。

**识别特征**：
- 非脚本化的真实反应
- 语气、语速、音量的突然变化
- 观众能感受到"这是真话"的瞬间

**评分映射**：命中情绪峰值 → viralPotential +12

### 3. 观点炸弹 (OPINION BOMBS)

强烈、极化或反直觉的观点，触发观众"同意/不同意"的本能反应。

**识别特征**：
- 与主流观点相悖
- 足够极端以引发讨论
- 不是为了争议而争议，而是有论据支撑

**中文平台典型句式**：
- "读书无用论是错的，但比读书更没用的是..." / "努力是最不重要的成功因素"

**评分映射**：命中观点炸弹 → viralPotential +12

### 4. 揭示时刻 (REVELATION MOMENTS)

令人惊讶的事实、数据或坦白，重构观众的认知框架。

**识别特征**：
- 具体数字或统计数据
- 内部信息的首次公开
- "原来如此"的顿悟感

**评分映射**：命中揭示时刻 → viralPotential +10

### 5. 冲突/张力 (CONFLICT/TENSION)

分歧、反驳或问题被正面迎击的时刻。

**识别特征**：
- 访谈中的直接追问
- 不同观点的正面碰撞
- "但你之前说的跟这个矛盾"类挑战

**评分映射**：命中冲突/张力 → viralPotential +10

### 6. 可引用金句 (QUOTABLE ONE-LINERS)

可以独立作为引语卡片传播的一句话。

**识别特征**：
- 不依赖上下文即可理解
- 有修辞力度（对仗、隐喻、反差）
- 适合截图/引用/转发

**中文平台典型句式**：
- "你以为的极限，只是别人的起点" / "自由不是想做什么就做什么，而是不想做什么就可以不做什么"

**评分映射**：命中金句 → viralPotential +8

### 7. 故事高潮 (STORY PEAKS)

轶事或故事的高潮或反转——回报观众注意力的时刻。

**识别特征**：
- 叙事弧线的顶点
- 意料之外的转折
- "然后呢？"的悬念被释放

**评分映射**：命中故事高潮 → viralPotential +8

### 8. 实用价值 (PRACTICAL VALUE)

观众可以立即应用的具体技巧、方法或洞察。

**识别特征**：
- 可操作的步骤
- 具体的工具或资源推荐
- "看完就能用"的即时价值

**评分映射**：命中实用价值 → viralPotential +8

## 综合评分规则

- 命中 5+ 维度：viralPotential = 85-100（高传播力）
- 命中 3-4 维度：viralPotential = 65-84（中等传播力）
- 命中 1-2 维度：viralPotential = 40-64（低传播力）
- 命中 0 维度：viralPotential = 0-39（无传播力）
