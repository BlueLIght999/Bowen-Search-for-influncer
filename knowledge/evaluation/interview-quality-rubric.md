# 访谈质量评估规则

## 四维评分标准

### 1. 提问深度 (0-100)

| 分值区间 | 标准 | 示例 |
|---------|------|------|
| 80-100 | 问题具体、有上下文、引导深度回答 | "你在阿里最后一年负责的具体业务是什么？那个决定是在什么场景下做出的？" |
| 60-79 | 问题有针对性但不够具体 | "你怎么看待创业？" |
| 30-59 | 问题泛泛，缺乏方向 | "你觉得怎么样？" |
| 0-29 | 封闭式问题，只能回答是/否 | "你觉得难吗？" |

### 2. 开放性 (0-100)

**开放性问题关键词**：怎么看待、为什么、什么感受、什么想法、能说说、怎样的
**封闭性问题关键词**：是不是、有没有、会不会、好不好、难不难

- 开放占比 > 70% → 80-100
- 开放占比 50-70% → 60-79
- 开放占比 30-50% → 30-59
- 开放占比 < 30% → 0-29

### 3. 追问有效性 (0-100)

- **追问链**：后一个问题基于前一个回答的内容
- 3+ 个追问形成链条 → 70-100
- 2 个追问 → 40-69
- 1 个提问无追问 → 10-39
- 无提问 → 0

### 4. 节奏控制 (0-100)

- **均匀分布**：问题在时间线上均匀出现
- **问题聚类**：多个问题连续出现，中间无回答间隔
- 变异系数 < 0.5 → 80-100
- 变异系数 0.5-1.0 → 60-79
- 变异系数 > 1.0 → 30-59

## 改进建议映射

| 评分维度 | 阈值 | 建议方向 |
|---------|------|---------|
| 开放性 < 50 | 高优先级 | 将封闭式提问改为开放式提问 |
| 追问有效性 < 40 | 中优先级 | 基于受访者回答的关键词进行追问 |
| 结构完整度 < 50 | 中优先级 | 补充开场引入和金句收尾 |
| 节奏控制 < 50 | 低优先级 | 避免问题集中出现 |

## 收藏触发点识别规则

1. **金句句式**：包含"不是...是..."、"真正的...不是...而是..."句式
2. **反常识观点**：与大众认知相反的陈述
3. **实用清单**：可截图保存的操作建议
4. **情感高潮**：受访者真实情感爆发的片段
